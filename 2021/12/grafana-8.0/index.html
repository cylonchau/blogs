grafana v8.0+ 隐藏表格字段

2021-12-17 · 54 字 · 1 分钟  · 
    #Monitor

  1. Select panel title → Inspect → Panel JSON

  2. Set “type” to “table-old”

  3. Apply

  4. The visualization should now appear as Table (old) and in the right side will appear Column Styles

  5. Column Styles → Options → Name pattern set the name of the column to hide

  6. Type → Hidden

Reference：Hide column in table in v8.0
